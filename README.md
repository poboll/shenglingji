# Shenglingji (圣灵集)

A modern social media platform built with Android native and Node.js.

## Project Structure

- `shenglingji-android/`: Android native application
- `shenglingji-backend/`: Node.js backend server

## Features

- Modern UI with Material Design
- Real-time messaging
- Media sharing
- Social networking features
- User authentication and authorization

## Getting Started

### Backend Setup

```bash
cd shenglingji-backend
npm install
npm start
```

### Android Setup

```bash
cd shenglingji-android
./gradlew build
```

## Requirements

### Backend
- Node.js 16+
- npm or yarn
- MySQL 8.0+

### Android
- Android Studio Hedgehog | 2023.1.1
- JDK 21
- Android SDK 34
- Gradle 8.4
