快速上手

只需几步，即可在本地运行「生灵集」项目。

后端服务

后端采用 Node.js 和 Express 构建，负责提供 API 服务和数据处理。

# 1. 克隆仓库git clone https://github.com/poboll/shenglingji.git

# 2. 进入后端目录并安装依赖cd shenglingji/shenglingji-backend
npm install

# 3. 配置 .env 文件 (数据库等)cp .env.example .env

# 4. 启动服务npm start

安卓客户端

安卓端使用 Kotlin 和 Jetpack Compose 构建，提供现代化的原生体验。

# 1. 使用 Android Studio 打开项目File > Open... > (选择 'shenglingji/shenglingji-android' 目录)

# 2. 等待 Gradle 同步完成Android Studio 会自动下载所需依赖

# 3. 修改后端 API 地址在代码中找到 API 地址配置，修改为您的本地后端地址 (e.g., http://10.0.2.2:3000)

# 4. 运行应用点击 'Run' 按钮或使用快捷键 (Shift+F10)
